# HR Analytics Power BI Dashboard

## Tools Used
- Power BI Desktop
- Power Query
- DAX (Basic)
- Excel (HR Dataset)

## Project Overview
This project analyzes employee attrition and workforce trends using an interactive Power BI dashboard.
The dashboard helps HR teams understand key factors affecting employee retention.

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company

## Dashboard Analysis
- Higher attrition observed in certain departments.
- Employees with lower job satisfaction show higher attrition.
- Salary and experience levels impact employee retention.
- Attrition trends vary across age groups.

## Business Insights
- Improving job satisfaction can help reduce attrition.
- Targeted retention strategies can be applied department-wise.
- HR teams can use these insights for better workforce planning.
